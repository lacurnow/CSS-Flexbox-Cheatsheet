What is flexbox?

CSS Flexbox is a layout module that permits responsive, dynamic and flexible arrangement of elements without using float or positioning.

How does flexbox work?

To arrange elements using flexbox, elements are contained in a 'flex container'. The child elements within are called 'flex items', and these flex items are placed along 'flex lines'. Lines and items can be manipulated in layout, size and spacing and more along the x and y axes using operators. Elements can then arrange themselves automically in a responsive manner ('flexing'), allowing optimal utilisation of space. Operators in the flexbox module can act upon the parent container or its children.

When to use Flexbox

Flexbox is not recommended for building whole webpages (larger layouts may instead use Grid), but it is appropriate for small-scale layouts and applications e.g. nav menus, web forms. Flexbox Concepts & Terminology

Term

Concept
main axis The axis on which flex items are laid out. Can be vertical or horizontal, depending on the flex-direction property.
main-start, main-end, cross-start, cross-end Represent the start and end points of where items are arranged
main size, cross size Denotes either the width or height of the flex items, depending on orientation of main axis.
cross axis The axis perpendicular to the main axis.

Flexbox Properties

Property

Description

Options

Description
flex-direction Defines the main axis and therefore which direction the flex items are placed. Permits changing the order of items independently of where they are in the html.
Flex-flow is used as shorthand for this property. 		row By default; arranges items left to right.
row-reverse Arranges items right to left.
column Items arranged top to bottom.
column-reverse Displays items bottom to top.
flex-wrap This property permits specification of how items will be organised in a flex container.
Flex-flow is used as shorthand for this property. 		nowrap The default value; places all items in a row along one line within the flex container.
wrap If there are too many items to fit along a single line, this option will arrange items over multiple lines, from top to bottom.
wrap-reverse As wrap, but arranged bottom to top.
flex-flow Shorthand for flex-direction and flex-wrap. Allows both the main axes of the container to be defined; default values are row nowrap.
justify-content Defines item alignment on the main axis. Controls what happens with any available free space. Has some control over alignment when items get wider than their container. Browser support for these values is not consistent so be wary. flex-start, flex-end Default. Items align towards the front / end of flex-directionrespectively.
start Defines the beginning of writing-mode as the starting point, or moves items towards the end of it.
left, right Aligns flex items towards left / right edge of container, or will order items like start depending on the specification of flex-direction
center Will order items horizontally.
space-between Items will be distributed evenly within the container. The first towards the start, the last towards the end and even space between each item.
space-around Items are evenly spaced with the same amount of space around each item (double the space between items).
space-evenly Items sit evenly spaced within the container but the spacing between each item, and towards the container edges is even.
